# Documentation

This folder contains documentation for the INDI CHARTS application.

## Privacy Policy

The Privacy Policy is available at: `privacy.md`

### Setting up GitHub Pages

To make the Privacy Policy accessible via GitHub Pages:

1. **Push the `docs` folder to your repository**
   ```bash
   git add docs/privacy.md
   git commit -m "Add Privacy Policy"
   git push
   ```

2. **Enable GitHub Pages**:
   - Go to your GitHub repository
   - Click **Settings** → **Pages**
   - Under **Source**, select:
     - Branch: `main` (or your default branch)
     - Folder: `/docs`
   - Click **Save**

3. **Access your Privacy Policy**:
   - URL format: `https://your-username.github.io/repo-name/privacy`
   - For example: `https://username.github.io/rsi-widget/privacy`

4. **Use in Google Play Console**:
   - Copy the Privacy Policy URL
   - Go to Google Play Console → Your App → App content → Privacy Policy
   - Paste the URL and save

### Alternative: Direct File Link

You can also use the direct GitHub file link:
- Format: `https://github.com/your-username/repo-name/blob/main/docs/privacy.md`
- Note: Some platforms prefer `.md` files, others prefer `.html`

If you need an HTML version, you can convert the markdown file using any markdown-to-HTML converter.
